허용 가능한 1종 오류 최대 한계
→ 1종↔2종 오류 사이의 관행적 균형점

의사결정실제 H₀ 참 (차이 없음)실제 H₀ 거짓 (차이 있음)

없는데 "있다"고 잘못 결론		✅ 올바른 결정
검정력 = 1−β
p ≥ 0.05 → H₀ 채택✅ 올바른 결정❌ 2종 오류 (β)
있는데 "없다"고 놓침

p ≥ 0.05 → 우연으로도 설명 가능 → 귀무가설 기각 실패
📏신뢰구간 (Confidence Interval, CI)9주차 핵심

95% CI의 의미

같은 방식으로 100번 표본을 뽑으면
그 중 95번은 모수가 이 구간 안에 포함됨
→ "모수가 이 구간에 있을 확률 95%"가 아님!

CI 공식 (단일표본)

CI = x̄ ± z* × (σ/√n)
→ σ알려짐: z* = 1.96 (95%)
→ σ모름: t* 사용 (df = n−1)

상황 공식 해석
단일표본 (σ알려짐)x̄ ± 1.96·σ/√nz분포 사용
단일표본 (σ모름)x̄ ± t*(df)·s/√nt분포 사용, df=n−1
CI와 가설검정 관계μ₀ ∉ CI → p<0.05CI에 귀무값 미포함 = 유의
OR/HR 95% CICI에 1 포함 → 유의하지 않음비율 지표는 귀무값=1
🧮검정통계량 공식 — 각 검정별 t/z/F/χ² 계산계산 기준
검정법검정통계량 공식분포 / 자유도기각역
z-test
 z = (x̄ − μ₀) / (σ/√n) Z ~ N(0,1)|z| > 1.96
Independent t
 t = (x̄₁−x̄₂) / Sp√(1/n₁+1/n₂)
Sp = 합동표준편차 (pooled SD)		 t(n₁+n₂−2)p < 0.05
Paired t
 t = d̄ / (Sd/√n)
d = 각 쌍의 차이, d̄ = 차이의 평균		 t(n−1)p < 0.05
One-way ANOVA
 F = MSB / MSW
MSB=SSB/(k−1), MSW=SSW/(N−k)		 F(k−1, N−k)p < 0.05
χ² 연관성
 χ² = Σ (O−E)² / E
O=관찰빈도, E=기대빈도		 χ²((r−1)(c−1))p < 0.05
McNemar
 χ² = (b−c)² / (b+c)
b=전(+)후(−), c=전(−)후(+)		 χ²(1)p < 0.05
💡 F통계량 직관: F = 집단 간 분산 / 집단 내 분산. F가 클수록 집단 간 차이가 집단 내 변동보다 크다 → 귀무가설 기각 근거 강해짐

🎯자유도 (Degrees of Freedom, df) — 검정별 계산법핵심 개념
💡 자유도란? — "데이터에서 자유롭게 변할 수 있는 값의 수"
직관적 이해: n개의 데이터로 평균을 구하면 → 마지막 1개 값은 자동으로 결정됨 (나머지를 이미 알면 마지막은 "자유롭지 않음")
→ n개 데이터, 1개 추정 파라미터(평균) → 자유도 = n − 1
자유도가 클수록 → 분포가 정규분포에 가까워짐 → 임계값이 작아짐 → 귀무가설 기각이 쉬워짐
① t검정 자유도
단일표본 t / 대응표본 t
df = n − 1
독립표본 t검정
df = n₁ + n₂ − 2
② 카이제곱 자유도
단일표본 카이제곱
df = 범주 수 − 1
연관성 카이제곱 (교차표)
df = (행 − 1) × (열 − 1)
③ ANOVA 자유도 — F(집단 간 df, 집단 내 df)
변동 종류 df 공식 의미 예시 (k=3, 각 n=10)
집단 간 (Between) k − 1 집단 수 k 3 − 1 = 2
집단 내 (Within) N − k 전체 N, 집단 수 k 30 − 3 = 27
전체 (Total) N − 1 집단간 + 집단내 30 − 1 = 29
F(2, 27) = 집단 간 분산 / 집단 내 분산  →  임계값 3.36 @ α=0.05
📌 자유도 크기 → 임계값 변화 → 귀무가설 기각 난이도
💡 핵심 요약: 자유도 = 표본크기(n)에 의존 → 표본이 클수록 df 증가 → t분포·χ²분포가 정규분포에 가까워짐 → 더 작은 차이도 통계적으로 유의하게 탐지 가능
📉모델 성능 평가: 혼동행렬 · 성능지표 · ROC · AUC10주차 1차시 · 12주차 1차시
① 혼동행렬 (Confusion Matrix) — 구조와 각 셀의 의미
② 5가지 성능지표 — 공식과 의미
지표 공식 의미 / 관심사 Model 1 예시
정확도 Accuracy (TP+TN) / (TP+FN+FP+TN) 전체 예측 중 맞은 비율 (142+110)/303 = 0.83
민감도 Sensitivity
(= Recall = TPR)		 TP / (TP + FN) 실제 양성 중 올바르게 잡아낸 비율
관심: 실제 질환 있는 환자		 142/(142+22) = 0.87
특이도 Specificity
(= TNR)		 TN / (TN + FP) 실제 음성 중 올바르게 잡아낸 비율
관심: 실제 질환 없는 환자		 110/(110+29) = 0.79
정밀도 Precision
(= PPV)		 TP / (TP + FP) 양성 예측 중 실제 양성인 비율
관심: 예측의 신뢰도(품질)		 142/(142+29) = 0.83
F1 score 2 × (Precision×Recall) / (Precision+Recall) 정밀도와 민감도의 조화평균
둘 다 골고루 높을 때 커짐		 2×(0.83×0.87)/(0.83+0.87) = 0.85
FN 최소화가 중요한 경우
→ 민감도(Recall) ↑ 중시
예) 결핵·HIV 검사, 갑작스런 사망 질환
FN: "있는데 없다"고 해서 치료 기회 놓침

🎲로지스틱 회귀 — 오즈비(OR) 해석 완전 정리13주차 2차시
📐 오즈(Odds)와 오즈비(OR) 개념

오즈(Odds)란?

Odds = p / (1−p)
사건 발생 확률 / 미발생 확률
예) 발병률 25% → Odds = 0.25/0.75 = 0.33
→ "발생하지 않을 확률의 0.33배 발생"

오즈비(OR)란?

OR = Odds₁ / Odds₀
노출군 오즈 ÷ 비노출군 오즈
로지스틱 회귀에서 exp(β)로 계산
→ 두 집단의 상대적 오즈 비교

🔑 OR 값에 따른 해석
OR 값 의미 예시 95% CI 해석
OR > 1위험 증가흡연자 폐암 OR=3.5 → 흡연자가 비흡연자보다 폐암 오즈 3.5배CI가 1 초과이면 유의
OR = 1차이 없음노출 여부가 결과에 영향 없음CI가 1 포함 → 유의하지 않음
OR < 1보호 효과채소 섭취 OR=0.6 → 채소 섭취군이 비섭취군보다 오즈 40% 감소CI가 1 미만이면 유의
💻 R 로지스틱 회귀 output 해석
glm(disease ~ smoking + age, family=binomial)

Coefficients:
          Estimate  Std.Error  z value  Pr(>|z|)
(Intercept) -4.210    0.852     -4.941   <0.001 ***
smoking     1.253     0.331      3.785    0.0002 ***
age         0.048     0.012      4.000    <0.001 ***
OR 계산: exp(Estimate) 로 변환
→ smoking: exp(1.253) = 3.50 → 흡연자가 비흡연자보다 질환 오즈 3.5배
→ age: exp(0.048) = 1.049 → 나이 1세 증가 시 오즈 4.9% 증가
R에서: exp(coef(model)) 또는 exp(confint(model))로 OR과 95% CI 동시 출력
⚠️ OR vs RR(상대위험도) 차이: OR은 오즈의 비율, RR은 확률의 비율. 사건이 드물 때(희귀질환, 발생률 <10%) OR ≈ RR이므로 혼용 가능. 발생률이 높으면 OR이 RR을 과대 추정!
🧬생존분석 · 카플란-마이어 · Cox 비례위험모형13~14주차
카플란-마이어(KM) 생존곡선

계단 형태로 시간에 따른 생존율 표시
+ 표시 = 중도절단 (사건 미발생으로 추적 종료)
Log-rank test로 두 곡선 차이 검정

Cox 비례위험모형 — HR(위험비) 해석

HR < 1: 위험 감소 (예: 커피 → 심방세동 재발 0.61배)
HR > 1: 위험 증가 (예: 고령 → 사망위험 1.42배)
95% CI가 HR=1 포함하면 유의하지 않음

🔑 KM + Log-rank vs Cox 비례위험모형 — 언제 무엇을 쓰나?
Kaplan-Meier + Log-rank: 시각적으로 직관적 | 이분형 변수 1개만 처리 | 집단 간 생존곡선 비교
Cox 비례위험모형: 다변량 분석 가능 | 연속형 변수 그대로 투입 | HR(위험비)로 각 변수 기여도 해석

비례위험 가정 확인 (Schoenfeld residual test / cox.zph):
→ 각 predictor별 p > 0.05 → 가정 만족 → Cox 모델 정당화
→ p < 0.05 → 가정 위반 → 해당 변수 제거 또는 다른 모델 고려
방법변수 수변수 형태결과 표현단점
